Interactive Task Manager

A simple and professional Task Manager web app built with **HTML, CSS, and JavaScript**.

  Features
- Add, complete, and delete tasks
- Filter tasks: All / Completed / Pending
- Stores tasks in **localStorage**
- Loads initial tasks from a **REST API**
- User-friendly error handling and loading indicators
- Keyboard accessibility (Enter to add task)
- Clean and modern UI

  Technologies
HTML | CSS | JavaScript | localStorage | REST API
